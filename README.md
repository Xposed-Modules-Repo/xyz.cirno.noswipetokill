# NoSwipeToKill

禁用 MIUI 划卡即强杀应用行为<del>，又名你杀我 VPN 干啥</del>

## 兼容性

仅在 MIUI 14 上测试通过。

如果在其他版本上正常工作，欢迎前往[源码仓库](https://github.com/dantmnf/NoSwipeToKill/issues)回报。

## 配置

| 在最近任务中上锁 | 或后台策略设置为无限制 |
